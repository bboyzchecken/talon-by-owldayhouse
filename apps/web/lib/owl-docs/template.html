ลายเซ็น & ท้ายเอกสาร

ค่าเริ่มต้นเป็นโลโก้นกฮูก ODH ตัวจริง หากต้องการโลโก้แบบเต็ม (มีข้อความ) ให้วาง URL ไฟล์โลโก้ของบริษัท

วิธีให้ออกมาตรงเป๊ะเป็น PDF: กด พิมพ์ / บันทึก PDF → เลือกเครื่องพิมพ์เป็น Save as PDF → ตั้งกระดาษ A4 และ Margins = None (ระยะขอบ = ไม่มี) → Save.  ปุ่ม "ตัวอย่าง" ด้านซ้ายโหลดข้อมูลจากเอกสารต้นแบบทั้ง 3 ใบให้ดูเทียบได้ทันที
